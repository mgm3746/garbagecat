Provides OpenJDK and Sun JDK domain classes.

Common handling of logging events:

1. Ignore any whitespace at the end of a logging line.

2. Ignore the "Times" block that is sometimes at the end of a logging line. For example:

251.763: [GC [1 CMS-initial-mark: 4133273K(8218240K)] 4150346K(8367360K), 0.0174433 secs] [Times: user=0.02 sys=0.00, real=0.02 secs]

3. Ignore icms_dc data. For example:

165.805: [Full GC 165.805: [CMS: 101481K->97352K(1572864K), 1.1183800 secs] 287075K->97352K(2080768K), [CMS Perm : 68021K->67965K(262144K)] icms_dc=10 , 1.1186020 secs]
